# Breve resumo da Sprint:
## Nessa sprint tivemos que resolver alguns exercícios, como por exemplo um exercício de geração de dados, gerando nomes e animais aleatórios e adicionando cada um para um arquivo, os nomes para txt e os animais para csv, e um outro que adicionamos alguns dados aos nomes, criando outras colunas, como "escolaridade" e "país", além desses dois exercícios, também puxei o exercício do TMDB da sprint passada como foi recomendado.
## Sobre o desafio, demos continuidade ao que foi feito na sprint passada, pegando tanto os dados do csv, quanto dos json que fizemos na sprint passada, puxando pela api, transformando ambos em parquet, utilizando os jobs do AWS glue, o crawler e o AWS athena para verificar se os parquet foram criados corretamente.
